SERVIÇO DE AJUDA AO CLIENTE.

Rest é uma solução de software para automação de dispositivos em casas inteligentes.

Sua navegação é feita de forma bem simples.

Primeiro, é necessário efetuar o cadastro no Rest.

Ao realizar e validar o cadastro a solução já estará disponível para uso imediato.

Após o cadastro você deve escolher entre as opções de uso na barra de tarefas localizada no canto superior da tela. Com base nessa seleção é possível navegar pelo programa e utilizar suas funcionalidades.

Recomendamos que tire um tempo para analisar todas as opções oferecidas.

Para poder controlar seus dispositivos, basta cadastra-los na nossa plataforma, identificando o dispositivo, onde ele se encontra e quem terá acesso a ele.

Agora é só começar a usar e aproveitar o conforto e comodidade que Rest trás para você!
